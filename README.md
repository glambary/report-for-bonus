I was asked to write a program for compiling an Excel document, namely calculating the premium, according to a given algorithm, because this work is very monotonous and takes a lot of time.
Program algorithm:
1) The program finds the month you are interested in in the file (by default zp.xlsx ). The desired month and the name of the files can be set in the example file.xlsx
2) There is a passage and verification, according to the selected lines:
   - whether the documents we are interested in are in the on file.xlsx (default);
   - and whether the amounts match, because the order can be partially shipped.
Used libraries: OpenPyXl, PyInstaller (for conversion to exe).
